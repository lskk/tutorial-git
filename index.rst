Tutorial Belajar Git dan GitHub untuk Pemula
============================================

Yuk belajar kolaborasi proyek berbasis *version control system*, sarana penting bagi pelajar, guru, mahasiswa, dosen, peneliti, profesional, dan *software developer* yang ingin berkolaborasi dalam tim. Kursus online ini akan berfokus pada Git, sebuah *version control system* populer, dan platform kolaborasi `GitHub <https://www.github.com/>`_.

Dokumentasi kursus online ini ada di https://tutorial-git.readthedocs.io/id/latest/, yang merupakan hasil *build* otomatis dari repository GitHub dokumentasi kolaboratif di https://github.com/lskk/tutorial-git.
